Ejercicios condicionales

Ejercicio 1: Ano bisiesto

Crea un programa que verifique si un año es bisiesto o no. Un año es bisiesto si:

Es divisible por 4. No es divisible por 100, a menos que también sea divisible por 400. Requisitos:

Solicitar al usuario que ingrese un año. Utilizar booleanos para realizar las comprobaciones necesarias. Mostrar un mensaje indicando si el año es bisiesto o no.

Ejercicio 2: Verificacion de vocal

Crea un programa que verifique si una letra ingresada por el usuario es una vocal (a, e, i, o, u) o no. El programa debe:

Solicitar al usuario que ingrese una letra. Utilizar un booleano para verificar si la letra es una vocal. Mostrar un mensaje indicando si la letra es una vocal o no

Ejercicio 3: Verificación de Rango de un Número

Crea un programa que verifique si un número ingresado por el usuario se encuentra dentro de un rango específico (por ejemplo, entre 10 y 50). El programa debe:

Solicitar al usuario que ingrese un número. Utilizar booleanos para verificar si el número está dentro del rango. Mostrar un mensaje indicando si el número está dentro o fuera del rango.

Ejercicio 3: Calificaciones de estudiantes

Eres un profesor y tienes las calificaciones de 30 estudiantes en una asignatura. Necesitas calcular el promedio de calificaciones y ver cuántos estudiantes aprobaron.
